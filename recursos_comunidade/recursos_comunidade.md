# Recursos da Comunidade para o Exame DP-700

Além dos recursos oficiais da Microsoft, a comunidade de dados produz um conteúdo valioso para ajudar na preparação para a certificação. Esta seção compila vídeos do YouTube, blogs, guias de estudo e simulados de alta qualidade criados pela comunidade.
